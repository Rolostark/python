# Curso de Python: Sintaxis Básica

## 1. Variables

Las variables almacenan datos. No necesitas declarar el tipo de variable en Python.

```python
# Ejemplo de variables
nombre = "Juan"
edad = 25
altura = 1.75
```

## 2. Tipos de Datos

- **int**: Números enteros  
- **float**: Números decimales  
- **str**: Cadenas de texto  
- **bool**: Booleanos (True/False)

```python
numero_entero = 10        # int
numero_decimal = 3.14     # float
texto = "Hola, mundo"     # str
es_mayor = True           # bool
```

## 3. Operadores

### Aritméticos

```python
a = 10
b = 3

suma = a + b        # 13
resta = a - b       # 7
multiplicacion = a * b  # 30
division = a / b    # 3.333...
modulo = a % b      # 1
potencia = a ** b   # 1000
```

### Comparación

```python
a = 5
b = 7

print(a == b)   # False
print(a != b)   # True
print(a < b)    # True
print(a > b)    # False
print(a <= b)   # True
print(a >= b)   # False
```

### Lógicos

```python
x = True
y = False

print(x and y)  # False
print(x or y)   # True
print(not x)    # False
```

## 4. Imprimir en pantalla

```python
nombre = "Ana"
print("Hola,", nombre)
```
